Bittool

Command line tools to access Bitcasa service.

Download this project as a .zip file Download this project as a tar.gz file

Bittoolとは

Bitcasa上のファイル、ディレクトリにコマンドラインからアクセスするツールです。

アクセストークン取得

Bittoolを使うには、アクセストークンを取得する必要があります。アクセストークン取得からログインするとアクセストークンが発行されますので、ファイルに保存します。
